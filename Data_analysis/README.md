## 데이터 분석을 위한 라이브러리



### 파이썬의 여러가지 모듈과 패키지



- 패키지 ?

  > 모듈.py 파일들이 모인 디렉토리의 집합

- 모듈 ?

  > 특정 목적을 가진 함수, 자료가 모인 py 파일

- 파이썬에서 불러오기

  > 예) 계산기 패키지를 만들었다.
  >
  > cal01 패키지 안에 cal.py 모듈을 만들고 안에 여러가지 함수가 있을 경우

  ```python
  from cal01.cal import mul # 특정 함수만 불러오기
  ```
